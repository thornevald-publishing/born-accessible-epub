ONIX Metadata Matters Too

Compliance isn’t just about the EPUB file.

ONIX accessibility metadata communicates your book’s accessibility status before it reaches storefronts or libraries.

Publishers should include:

  • Accessibility features
  • Known limitations
  • Accessibility summary or conformance codes

Retailers, libraries, and discovery systems use ONIX to:

  • Display accessibility filters
  • Flag compliant titles
  • Influence procurement or discoverability

EPUB metadata ensures the file itself is accessible. ONIX metadata ensures the market knows it is.
